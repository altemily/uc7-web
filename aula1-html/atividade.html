O que sãos TAGs HTML?

Definição:

A expressão “tag” significa literalmente “etiqueta”. Uma etiqueta é usada para marcar ou identificar algo, como a etiqueta de preços no mercado por exemplo. No HTML5, as tags tem o mesmo objetivo: marcar elementos. Ou seja, determinar onde um elemento começa e onde ele termina.

Forma de uso:

Basta colocar a tag no cabeçalho da sua página HTML e escrever o seu código dentro dela. Geralmente, elas são usadas para adicionar CSS e JavaScript a um documento HTML. O CSS é usado para estilizar o conteúdo do documento, enquanto o JavaScript é executado pelo navegador para adicionar funcionalidades ao documento.
